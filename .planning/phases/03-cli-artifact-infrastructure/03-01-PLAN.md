Replace the stub `longmemeval.rs` with the full CLI foundation. Follow the LoCoMo locomo.rs pattern exactly for structure. All types and logic go in this one file (matching LoCoMo's single-file pattern per research anti-patterns).

**Enums:**
- `BenchCommand` { Run, Ingest, Qa } with `as_str()`. No Merge (deferred to v2).
- `RunProfile` { Smoke, FullS, FullM } with `as_str()`, `config_path()` -> `bench/longmemeval/profiles/{name}.json`, `FromStr`, `Default` = FullS.

**Config structs (adapted from LoCoMo):**
- `FileRunConfig` (all-optional serde struct for profile JSON): dataset, output, tag, instances, session_limit, instance_limit, ingest_format (Option of IngestFormat from ingest.rs), consolidation (Option of ConsolidationMode from ingest.rs), instance_jobs, judge_model. Has `apply(&self, &mut RunConfig)`.
- `RunConfig` (concrete resolved config): profile, config_path, dataset (PathBuf), output (Option), tag (Option), instances (Vec<String>), session_limit (Option<usize>), instance_limit (Option<usize>), ingest_format (IngestFormat), consolidation (ConsolidationMode), instance_jobs (usize, default 1), judge_model (Option<String>), allow_overwrite (bool). Default: dataset=`data/longmemeval_s_cleaned.json`, ingest_format=Text, consolidation=End.

**CLI structs:**
- `CliOverrides` (mirrors RunConfig but all optional): same fields as FileRunConfig plus profile, allow_overwrite, help. Has `apply(self, &mut RunConfig)`.
- `ParsedCli` { command, artifact_path, overrides }.

**Functions:**
- `parse_args_from(args: &[String]) -> Result<ParsedCli, String>`: Manual arg parsing matching LoCoMo pattern. Subcommand is args[1]. For `qa`, args[2] is artifact path (required, not starting with `-`). Then flag loop: --profile, --config, --dataset, --out, --tag, --instance (repeatable), --session-limit, --instance-limit, --ingest-format, --consolidation, --instance-jobs, --judge-model, --force, --help.
- `parse_args() -> BenchInvocation`: Calls `parse_args_from(&env::args().collect())`, handles help/errors with process::exit.
- `resolve_fresh_config(overrides: CliOverrides) -> Result<RunConfig, String>`: Profile defaults -> config JSON overlay -> CLI flags. Three-layer resolution.
- `resolve_qa_config(artifact_path: &Path, overrides: CliOverrides) -> Result<RunConfig, String>`: Load artifact -> extract config -> apply restricted overrides.
- `validate_qa_overrides(overrides: &CliOverrides) -> Result<(), String>`: Reject --profile, --config, --dataset, --session-limit, --instance-limit, --ingest-format, --consolidation.
- `load_json_config(path: &Path) -> Result<FileRunConfig, String>`: Read + parse JSON file.
- `default_output_path(command, config, artifact_path) -> PathBuf`: Match LoCoMo's pattern. Default dir: `bench/longmemeval/results/local/`. Stem: tag if set, else `{profile}-{command}`. Qa without tag returns artifact path.
- `parse_usize_arg(raw, flag) -> Result<usize, String>`: Helper.
- `print_help()`: Usage message listing subcommands and flags.
- `fn main()` — stub that calls parse_args(), prints config summary, then "pipeline not yet wired (Plan 02)". Does NOT exit(1) — it should be a valid parse-and-print run.
