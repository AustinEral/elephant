**5. Add helper functions** (place near the bottom, before main, following locomo's pattern):

- `read_jsonl_records<T: DeserializeOwned>(path: &Path) -> Result<Vec<T>, String>` — read file, split lines, deserialize each
- `write_jsonl_records<T: Serialize>(path: &Path, values: &[T]) -> Result<(), String>` — initialize file, append each via `append_jsonl`

- `merge_source_artifact(bundle: &LoadedArtifactBundle) -> SourceArtifact` — extract path, fingerprint, mode, tag, commit
- `ensure_merge_compatible(base: &LoadedArtifactBundle, other: &LoadedArtifactBundle) -> Result<(), String>` — use `ensure_same!` macro to check: benchmark, judge_model, retain_model, reflect_model, embedding_model, reranker_model, consolidation_strategy, manifest protocol_version, manifest mode, manifest dataset_fingerprint, manifest ingest_format, manifest consolidation_strategy, manifest session_limit, manifest instance_limit. Also compare prompt_hashes and runtime_config via serde_json::to_string equality.
- `merge_profile_value(bundles: &[LoadedArtifactBundle]) -> String` — collect profiles into BTreeSet, return single or "mixed"
- `merge_concurrency_value<F>(bundles: &[LoadedArtifactBundle], select: F) -> usize` — if all same return it, else 0
- `warn_if_mixed<F>(bundles: &[LoadedArtifactBundle], label: &str, select: F)` — eprintln if mixed values
- `merge_artifacts(input_paths: &[PathBuf], output_path: &Path, tag: Option<String>) -> Result<(), String>` — the main merge function:
  - Load all bundles, call `ensure_merge_compatible` for each against base
  - `warn_if_mixed` for profiles, instance concurrency, source commits, dirty-tree state
  - Iterate bundles: collect question_ids into BTreeSet for overlap detection, check duplicate bank_ids, collect merged_results + merged_debug, accumulate stage_metrics via MetricsCollector, sum total_time_s
  - Sort merged_results by `question_id`, merged_debug by `question_id`
  - Write sidecar files via `write_jsonl_records`
  - Clone base manifest, set mode="merge", profile=merge_profile_value, command=env args joined, selected_instances=all seen question_ids, instance_concurrency=merge_concurrency_value, dirty_worktree=git_dirty_worktree(), source_artifact=None, source_artifacts=bundles.iter().map(merge_source_artifact)
  - Build BenchmarkOutput and write as pretty JSON
  - Print summary
